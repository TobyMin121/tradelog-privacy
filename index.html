1. 개인정보 처리 목적

TradeLog는 아래 목적으로만 개인정보를 처리합니다. 수집된 정보는 다음의 목적 이외의 용도로 사용되지 않습니다.

목적세부 내용
매매 기록 관리사용자가 직접 입력한 거래 내역, 손익, 전략을 기기에 저장하여 트레이딩 일지 기능을 제공합니다.
거래소 연동사용자가 입력한 API 키를 통해 거래소에서 거래 내역을 가져와 기기에 저장합니다.
심리 & 목표 관리감정 트래커, 목표, 매매 규칙 등 사용자 입력 데이터를 기기에 저장합니다.

2. 수집하는 정보

TradeLog는 다음 정보를 처리합니다. 모든 정보는 사용자가 직접 입력하며, 기기 외부(개발자 서버)로 전송되지 않습니다.

종류내용저장 위치필수 여부
거래소 API 키 바이낸스, OKX, 업비트, 빗썸 API Key / Secret Key iOS Keychain (암호화) 선택
매매 기록 코인 심볼, 방향, 진입·청산 가격, 손익, 전략, 메모 기기 로컬 저장소 선택
감정 기록 날짜별 트레이딩 심리 상태 기기 로컬 저장소 선택
목표 · 규칙 · 전략 사용자 정의 목표, 매매 원칙, 전략 노트 기기 로컬 저장소 선택
기본 자금 사용자가 입력한 초기 투자 자금 금액 기기 로컬 저장소 선택

⚠️ 위 모든 항목은 선택 사항입니다. 입력하지 않아도 앱의 기본 기능은 이용 가능합니다.

3. 수집하지 않는 정보

이름 / 이메일 전화번호 위치 정보 광고 식별자 (IDFA) 기기 고유 번호 사용 분석 데이터 충돌 리포트 연락처 · 카메라 생체 정보

TradeLog는 광고를 표시하지 않으며, 사용자 행동을 추적하거나 분석하지 않습니다. 회원가입이나 로그인이 없으므로 계정 관련 정보도 일절 수집하지 않습니다.

4. 데이터 저장 및 보호

🔒 API 키 — iOS Keychain 암호화 저장
✅
하드웨어 수준 암호화
iOS Keychain은 애플 Secure Enclave를 통해 보호됩니다.
✅
iCloud 백업 제외
API 키는 iCloud 자동 백업에 포함되지 않습니다. 기기에만 존재합니다.
✅
개발자 서버 전송 없음
API 키는 해당 거래소 서버에만 직접 전송됩니다. TradeLog 개발자 서버는 존재하지 않습니다.
📱 매매 기록 등 — 기기 로컬 저장
✅
기기에만 저장
매매 기록, 감정 기록, 목표 등 모든 데이터는 사용자의 스마트폰 내부에만 저장됩니다.
✅
네트워크 전송 없음
매매 기록은 인터넷을 통해 전송되거나 서버에 저장되지 않습니다.

6. 거래소 API 연동 안내

거래 내역 가져오기 기능은 사용자가 입력한 API 키로 각 거래소 서버에 직접 요청을 보냅니다. 이 과정에서 TradeLog 개발자 서버를 경유하지 않습니다.

거래소전송 대상 서버전송 내용수신 내용
바이낸스fapi.binance.com서명된 API 인증 요청거래 내역 (기기에 저장)
오렌지X (OKX)www.okx.com서명된 API 인증 요청거래 내역 (기기에 저장)
업비트api.upbit.comJWT 인증 요청주문 내역 (기기에 저장)
빗썸api.bithumb.com서명된 API 인증 요청거래 내역 (기기에 저장)
읽기 전용 API 권장 출금 권한 사용 불가 매매 실행 권한 사용 불가

각 거래소의 개인정보처리방침은 해당 거래소 홈페이지를 확인하시기 바랍니다.

7. 제3자 제공

TradeLog는 수집한 개인정보를 어떠한 제3자에게도 제공하지 않습니다.

광고 네트워크 제공 ✗ 분석 서비스(Firebase 등) ✗ 마케팅 활용 ✗ 제3자 SDK 미포함 ✗

단, 사용자가 직접 선택한 거래소의 API 서버로 인증 요청이 전송되는 것은 서비스 제공을 위한 불가피한 과정이며, 이는 제3자 제공에 해당하지 않습니다.

8. 정보주체의 권리

사용자(정보주체)는 개인정보보호법에 따라 다음 권리를 행사할 수 있습니다.

행사 가능한 권리
①
개인정보 열람 요구권
저장된 개인정보의 내용을 확인할 권리 — 앱 내 매매 기록, 설정에서 직접 확인 가능합니다.
②
개인정보 정정 요구권
잘못된 정보를 수정할 권리 — 앱 내에서 직접 수정 가능합니다.
③
개인정보 삭제 요구권
설정 → 모든 데이터 삭제, 또는 앱 삭제로 즉시 영구 삭제됩니다.
④
처리 정지 요구권
개인정보 처리를 중단할 권리 — API 키를 삭제하면 거래소 연동이 즉시 중단됩니다.
⑤
권리 침해 구제
개인정보 침해가 있다고 판단되는 경우, 개인정보보호위원회(privacy.go.kr) 또는 한국인터넷진흥원(118)에 신고할 수 있습니다.

💡 TradeLog는 모든 데이터가 기기에만 존재하므로, 위 권리의 대부분을 앱 내에서 직접 즉시 행사할 수 있습니다. 별도로 개발자에게 요청할 필요가 없습니다.

9. 개인정보 안전성 확보 조치

개인정보보호법 제29조에 따라 다음의 기술적·관리적 보호 조치를 취하고 있습니다.

🔐
암호화 저장
API 키 등 민감 정보는 iOS Keychain(Secure Enclave)을 통해 암호화 저장됩니다.
🌐
암호화 전송
모든 외부 통신은 HTTPS/TLS를 통해 암호화됩니다.
🚫
최소 권한 원칙
앱은 서비스 제공에 필요한 최소한의 정보만 처리하며, 불필요한 권한을 요청하지 않습니다.
🛡️
접근 통제
서버가 없으므로 외부 해킹 경로가 존재하지 않습니다. 데이터 접근은 기기 잠금 해제에 의해 보호됩니다.
☁️
클라우드 백업 제외
API 키는 iCloud 자동 백업에서 제외되어 클라우드를 통한 노출 위험이 없습니다.

10. 아동 보호

TradeLog는 만 14세 미만 아동을 대상으로 하지 않습니다. 금융 투자 관련 앱으로 성인 트레이더를 위해 설계되었습니다. 만 14세 미만 아동의 개인정보가 수집된 사실을 인지하게 될 경우, 즉시 해당 정보를 삭제하기 위한 조치를 취합니다.

11. 방침 변경

본 개인정보처리방침은 법령·정책 변경이나 서비스 변경 시 개정될 수 있습니다. 변경 사항은 아래와 같이 고지합니다.

  • 변경 내용은 이 페이지 상단 최종 업데이트 날짜에 반영됩니다.
  • 중요한 변경이 있을 경우, 앱 업데이트를 통해 별도 안내합니다.
  • 변경된 방침은 시행일로부터 적용됩니다.
