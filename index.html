최종 업데이트: 2025년 5월

📌 핵심 원칙: TradeLog는 서버가 없습니다. 모든 데이터는 오직 사용자의 기기에만 저장되며, 개발사 서버로 어떠한 개인정보도 전송되지 않습니다.

1. 수집하는 정보

TradeLog는 다음 정보를 수집합니다. 모든 정보는 사용자가 직접 입력하며, 기기 외부로 전송되지 않습니다.

종류 내용 저장 위치
거래소 API 키 바이낸스, OKX, 업비트, 빗썸 API Key / Secret Key iOS Keychain (암호화)
매매 기록 코인, 방향, 진입/청산가, 손익, 전략 메모 기기 로컬 저장소
감정 기록 날짜별 트레이딩 심리 상태 기기 로컬 저장소
목표 / 규칙 / 전략 사용자 정의 목표, 매매 원칙, 전략 노트 기기 로컬 저장소

2. 수집하지 않는 정보

이름 / 이메일 위치 정보 광고 식별자 (IDFA) 기기 고유 번호 사용 분석 데이터 충돌 리포트 연락처 / 카메라

TradeLog는 광고를 표시하지 않으며, 사용 패턴을 추적하거나 분석하지 않습니다.

3. 데이터 저장 및 보호

🔒 API 키 보호
✅
iOS Keychain 저장
하드웨어 수준 암호화 적용. 앱이 삭제되어도 안전하게 관리됩니다.
✅
iCloud 백업 제외
API 키는 클라우드 백업에 포함되지 않습니다.
✅
서버 전송 없음
API 키는 해당 거래소(바이낸스, OKX 등) 서버에만 직접 전송됩니다. TradeLog 서버는 존재하지 않습니다.

4. 거래소 API 연동 안내

거래 내역 가져오기 기능은 사용자가 입력한 API 키를 사용하여 각 거래소 서버에 직접 요청을 보냅니다.

  • 전송 대상: 바이낸스 (binance.com), OKX (okx.com), 업비트 (upbit.com), 빗썸 (bithumb.com)
  • 전송 내용: API 인증 정보 (서명된 요청)
  • 수신 내용: 해당 거래소의 거래 내역 (기기에만 저장)

각 거래소의 개인정보처리방침은 해당 거래소 웹사이트를 참고하시기 바랍니다.

읽기 전용 API 권장 출금 · 매매 권한 사용 불가

TradeLog는 읽기 전용 API 키 사용을 강력히 권장합니다. 읽기 전용 키는 잔고 조회 및 내역 조회만 가능하며, 출금이나 매매를 실행할 수 없습니다.

5. 제3자 공유

TradeLog는 사용자 데이터를 어떠한 제3자와도 공유하지 않습니다.

  • 광고 네트워크에 데이터 제공: ✗
  • 분석 서비스(Firebase, Mixpanel 등) 사용: ✗
  • 제3자 SDK 포함: ✗

6. 데이터 삭제

앱 내 설정 → 모든 데이터 삭제 기능으로 저장된 모든 데이터(매매 기록, API 키 포함)를 즉시 영구 삭제할 수 있습니다.

앱을 기기에서 삭제하면 로컬에 저장된 모든 데이터가 함께 삭제됩니다.

7. 아동 보호

TradeLog는 13세 미만 아동을 대상으로 하지 않습니다. 금융 투자 관련 앱으로, 성인 사용자를 위해 설계되었습니다.

8. 방침 변경

본 개인정보처리방침이 변경될 경우, 앱 업데이트를 통해 사용자에게 안내됩니다. 변경 사항은 이 페이지에 게시되며 최종 업데이트 날짜가 표시됩니다.
